Bug Tracker

Bug Tracker is a simple but useful application, your able to log bugs in testing as well as add notes to individual bugs. In a perfect world we wouldn't need this, but in development this can be your best friend. This has an old school feel to it, nothing to fancy I just wanted a useful tool to help me througout the development process. Multiple users can use this to track and fix bugs throughout an application. I utilized Vue.js, node.js, and Auth0 in this application.
